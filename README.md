# 🚀 智能数据可视化分析平台

一个基于Web的现代化数据可视化分析平台，集成了多种交互式图表类型、实时数据编辑、智能分析和高科技主题切换功能。

## ✨ 功能特性

### 📊 多样化图表类型
- **三角函数可视化** - 正弦和余弦函数的动态展示
- **电影票房排行** - 支持多种排序和图表类型切换
- **支付宝账单分析** - 饼图展示消费结构
- **汽车制动性能** - 速度与制动距离关系分析
- **英语成绩统计** - 班级成绩对比分析

### 🎨 高科技主题系统
- 🔷 **赛博朋克** - 蓝紫色渐变科技感
- 🌈 **霓虹灯光** - 活力四射的霓虹色彩
- ⚛️ **量子计算** - 青紫色的未来科技感
- 💎 **全息投影** - 蓝绿色系的透明质感
- 🌌 **极光幻境** - 深蓝渐变的神秘感

### 🛠️ 强大的编辑功能
- **实时数据编辑** - 直接修改数据并即时更新图表
- **动态图表切换** - 支持线形图、柱状图、饼图、散点图等多种类型
- **智能数据分析** - 自动生成数据洞察和统计摘要
- **参数调节** - 通过滑块和输入框实时调整图表参数

### 📤 数据导出功能
- **图表导出** - 将图表保存为PNG图片
- **数据导出** - 导出CSV格式数据
- **完整导出** - 导出JSON格式的完整数据集

### 📱 响应式设计
- 完美适配桌面端和移动端
- 流畅的动画过渡效果
- 现代化的毛玻璃效果界面

## 🚀 快速开始

### 环境要求
- 现代Web浏览器（Chrome、Firefox、Safari、Edge）
- 无需安装任何依赖

### 本地部署
1. 下载项目文件
2. 使用浏览器打开 `智能数据可视化分析平台.html`
3. 即可开始使用所有功能

### 在线访问
直接将HTML文件部署到Web服务器即可在线访问。

## 📖 使用指南

### 基础操作
1. **切换图表标签** - 点击顶部导航栏选择不同数据集
2. **主题切换** - 使用右上角下拉菜单切换主题
3. **图表类型** - 点击图表类型按钮切换可视化方式

### 数据编辑
1. **编辑按钮** - 点击各数据区的"编辑数据"按钮
2. **实时修改** - 在数据表格中直接修改数值
3. **自动更新** - 图表和分析会实时反映数据变化

### 高级功能
1. **参数调节** - 使用滑块调整函数参数
2. **数据筛选** - 通过下拉菜单筛选显示数据
3. **排序功能** - 选择不同的排序方式查看数据
